# IocfpemsP
 《化肥农药进销存管理系统专业版》是专门为化肥农药经销商而开发的一套管理软件，它可以记录化肥农药的进货、进货退货、销售、销售退货，查看库存及应收 应付等业务功能，专业版与普通版的区别是专业版中可以整进散出，是一款有效的管理软件。 该软件的功能主要包括： 1、基本信息（供货信息、种类品牌设置、往来客户信息、本单位信息）； 2、进货管理（进货登记、进货查询、进货退货、按供货商查询进货、按供货商查询退货）； 3、销售管理（销售登记、销售查询、销售退货、按客户名称查询进货、按客户名称查询退货）； 4、库存管理（库存明细查看、按种类品牌查询库存、按供货商名称查询库存） 5、财务管理（日常收支账（日常支出登记 、日常收入登记、日常收入查询、日常支出查询）、供货商账（还供商账登记、欠供商账统计）、客户账（客户还账登 记、客户欠账统计）、营业收支汇总（营业总收入统计、营业总支出统计））
